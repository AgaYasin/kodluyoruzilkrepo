# Insertion Sort Projesi
---
1. [22,27,16,2,18,6] dizisinin insertion sort aşamalarını yazınız.
```
1.[22,27|16,2,18,6] 
2.[16,22,27|2,18,6]
3.[2,16,22,27|18,6]
4.[2,16,18,22,27|6]
5.[2,6,16,18,22,27]
```
2. Big-o gösterimi
```
O(n^2)
```
3. Time Complexity
```
Best Case: O(n)
Average Case: O(n^2)
Worst Case: O(n^2)
```
4. Dizi sıralandıktan sonra 18 hangi case'e girer.
```
Average Case
```
